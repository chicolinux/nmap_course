## Project Overview

This is an introductory Nmap course repository from theaisec_academy. Content is in **Spanish**. The course teaches Nmap fundamentals for network scanning in cybersecurity contexts — installation, help documentation, and first scans. Aimed at beginners with basic command-line knowledge.

## Target audience
The target audience of this course is undergrads from Computer Science or Computer Engineering, IT Professionals who want to learn
how to take adventage of the network mapper tool

## Syllabus
This course of compose of 8 episodes, where episode 0 covers an introduction, installation and first scan.
Next seven episodes need to be incremental in complexity, covering the most important aspects of nmap.
The last two episodes should cover the NSE and how to enhance nmap with your own Lua scripts.

## Repository Structure

- `labs/` — hands-on lab exercises (currently empty, to be populated)
- `README.md` — Spanish-language course description

## Context

This is an educational/content repository, not a software project. There is no build system, test suite, or application to run. Work here consists of creating lab exercises, scripts, and documentation for students.

Labs should use safe, legal, authorized targets (e.g., `scanme.nmap.org`, local VMs, or purpose-built lab environments). Never include scans against real production systems or unauthorized targets in examples.
