# nmap_course
Aprende los fundamentos de Nmap, la herramienta esencial de escaneo de redes en ciberseguridad. En este curso introductorio instalarás Nmap, aprenderás a consultar su manual de ayuda y realizarás tu primer escaneo. Ideal para principiantes con nociones básicas de línea de comandos. ¡Empieza a explorar redes de forma segura y responsable!
